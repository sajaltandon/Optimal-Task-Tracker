## 🚀 Getting Started

### **Prerequisites**
- A modern web browser (Chrome 88+, Firefox 85+, Safari 14+, Edge 88+)
- Basic understanding of web technologies (optional)

### **Quick Start**
1. **Clone the repository:**
   ```bash
   git clone https://github.com/sajaltandon/Optimal-Task-Tracker.git
   cd Optimal-Task-Tracker
   ```

2. **Open in browser:**
   ```bash
   # Using Python (recommended)
   python3 -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Or simply open index.html in your browser
   open index.html
   ```

3. **Start organizing! 🎉**
   - Visit `http://localhost:8000`
   - Add your first task
   - Experience the beautiful animations
   - Enjoy productive task management!

## 🎯 Usage Guide

### **Adding Tasks**
1. Type your task in the input field
2. Press `Enter` or click "➕ Add Task"
3. Watch the smooth slide-in animation!

### **Managing Tasks**
- **Complete:** Click "✅ Done" to mark as completed
- **Undo:** Click "↩️ Undo" to restore completed tasks
- **Delete:** Click "🗑️ Delete" to remove tasks
- **Due Date:** Set optional due dates for better planning

### **Bulk Actions**
- **Clear All:** Use "🗑️ Clear All Tasks" to remove all tasks
- **Confirmation:** Safety dialog prevents accidental deletions


```css

    /* Modify timing and easing */
```
